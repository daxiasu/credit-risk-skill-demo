# 授信风险分析 Skill Demo

这是一个用于展示金融 SA-FDE Skill 原型的静态页面 Demo。

## Demo 说明

该页面模拟银行内部智能体平台中的“信贷经理工作台”：

- 左侧：信贷经理输入自然语言任务，并查看风险分析报告。
- 中间：展示后台 Router 如何命中 `credit-visit-risk-analysis` Skill，以及 Skill 的执行链路。
- 右侧：展示企业数据区，可修改关键变量观察风险关注等级变化。

## 重要声明

本 Demo 仅使用模拟企业数据。

本 Demo 不包含任何真实客户信息、银行内部数据、征信数据或商业秘密。

本 Demo 输出的是“走访前风险关注等级”，不构成授信审批、拒贷、定价或银行内部评级意见。

## 本地打开

直接打开 `index.html` 即可使用。

## GitHub Pages 发布

将本目录作为 GitHub 仓库根目录上传后，可在仓库设置中开启 GitHub Pages：

1. 进入 GitHub 仓库。
2. 打开 Settings。
3. 找到 Pages。
4. Source 选择 Deploy from a branch。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub 生成访问地址。

## 推荐演示路径

1. 选择一家企业，例如“宁波某智能装备有限公司”。
2. 点击“生成分析”，展示风险等级、风险线索、走访问题和补充材料。
3. 在右侧调整应收账款比例、现金流、被执行记录等变量。
4. 观察左侧报告和中间执行链路如何变化。
5. 强调 Skill 是业务规则、数据和模型执行流程的封装，而不是最终审批系统。
